============
Installation
============


Stable release
--------------

To install PIK3CA-mutation, run this command in your terminal:

.. code-block:: console

    $ pip install PIK3CA_mutation

This is the preferred method to install PIK3CA-mutation, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for PIK3CA-mutation can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/yjcyxky/PIK3CA_mutation

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/yjcyxky/PIK3CA_mutation/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/yjcyxky/PIK3CA_mutation
.. _tarball: https://github.com/yjcyxky/PIK3CA_mutation/tarball/master
